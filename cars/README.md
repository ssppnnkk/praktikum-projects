# Cars

## Задача
Создание модели для оценки рыночной стоимости автомобиля.
Ключевые параметры:
- качество предсказания;
- скорость предсказания;
- время обучения.

## Результат
Построена модель для определения рыночной стоимости автомобиля.

## Библиотеки
*pandas, sklearn, catboost, lightgbm*
